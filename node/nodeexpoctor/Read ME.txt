Open a Text Editor
 Nano (Simple Command-Line Editor):nano manage_node_exporter.py 
 Vim (Advanced Command-Line Editor):vim manage_node_exporter.py 

Paste the Script & Save the File

 In Nano:
 Press Ctrl+O to save.
 Press Enter to confirm the filename.
 Press Ctrl+X to exit.

 In Vim:
 Press Esc to enter command mode.
 Type :wq and press Enter to save and exit.

Make the Script Executable 

 chmod +x manage_node_exporter.py 


Run the Script:
python3 manage_node_exporter.py (or) sudo python3 manage_node_exporter.py
